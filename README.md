# 飲料店點餐管理系統 TeaShopOrderSystem

## 一、專案簡介

本專案是一套使用 C# Windows Forms 製作的飲料店點餐管理系統，主要模擬飲料店櫃台點餐流程。

系統提供飲料選擇、尺寸選擇、甜度選擇、冰塊選擇、加料選擇、購物車管理、完成訂單、菜單管理與歷史訂單查詢等功能。

本專案具備讀寫檔功能，會將飲料菜單資料儲存在 `drinks.csv`，並將完成的訂單紀錄儲存在 `orders.csv`。

---

- 開發工具：Visual Studio 2022
- 程式語言：C#
- 專案類型：Windows Forms App

## 三、主要功能

### 1. 點餐功能

使用者可以選擇：

- 飲料品項
- 尺寸
- 甜度
- 冰塊
- 加料
- 數量

選擇完成後可加入購物車，系統會自動計算單價、小計與總金額。

### 2. 購物車功能

購物車可以顯示目前點選的飲料資料，包含：

- 飲料名稱
- 尺寸
- 甜度
- 冰塊
- 加料
- 數量
- 單價
- 小計

使用者可以刪除選取項目，也可以清空整個購物車。

### 3. 完成訂單功能

按下「完成訂單」後，系統會自動產生訂單編號，並將訂單資料寫入 `orders.csv` 檔案中。

完成訂單後，購物車會自動清空，總金額也會重新歸零。

### 4. 菜單管理功能

管理者可以新增、修改、刪除飲料資料。

飲料資料包含：

- 飲料編號
- 飲料名稱
- 分類
- 基本價格
- 是否上架

按下「儲存菜單」後，系統會將菜單資料寫入 `drinks.csv`。

### 5. 歷史訂單查詢功能

系統可以讀取 `orders.csv`，並提供以下查詢功能：

- 顯示全部訂單
- 依日期查詢
- 依關鍵字查詢
- 顯示今日訂單數
- 顯示今日營業額

## 四、資料檔案說明

### 1. drinks.csv

`drinks.csv` 用來儲存飲料菜單資料。

欄位包含：

```csv
Id,Name,Category,BasePrice,Available
```

範例：

```csv
D001,珍珠奶茶,奶茶類,55,true
D002,紅茶,茶類,30,true
D003,綠茶,茶類,30,true
```

欄位說明：

| 欄位 | 說明 |
|---|---|
| Id | 飲料編號 |
| Name | 飲料名稱 |
| Category | 飲料分類 |
| BasePrice | 基本價格 |
| Available | 是否上架 |

---

### 2. orders.csv

`orders.csv` 用來儲存歷史訂單紀錄。

欄位包含：

```csv
OrderId,OrderTime,DrinkName,Size,Sugar,Ice,Toppings,Quantity,Subtotal,Total
```

範例：

```csv
O20260620153045,2026/06/20 15:30:45,珍珠奶茶,大杯,半糖,少冰,珍珠,2,150,150
```

欄位說明：

| 欄位 | 說明 |
|---|---|
| OrderId | 訂單編號 |
| OrderTime | 訂單時間 |
| DrinkName | 飲料名稱 |
| Size | 尺寸 |
| Sugar | 甜度 |
| Ice | 冰塊 |
| Toppings | 加料 |
| Quantity | 數量 |
| Subtotal | 小計 |
| Total | 訂單總金額 |

## 五、執行方式

1. 使用 Visual Studio 2022 開啟專案。
2. 開啟 `TeaShopOrderSystem.sln`。
3. 按下 `F5` 或點選上方「開始」執行程式。
4. 進入系統後即可使用點餐、菜單管理與歷史訂單查詢功能。

---

## 六、操作說明

### 1. 點餐流程

1. 進入「點餐」分頁。
2. 選擇飲料、尺寸、甜度、冰塊與加料。
3. 設定數量。
4. 按下「加入購物車」。
5. 系統會將飲料加入右側購物車表格。
6. 確認購物車內容與總金額。
7. 按下「完成訂單」。
8. 系統會產生訂單編號，並將訂單寫入 `orders.csv`。

---

### 2. 購物車操作流程

1. 加入飲料後，購物車會顯示目前點選的品項。
2. 若要刪除某一筆資料，可以先點選購物車中的該筆資料。
3. 按下「刪除選取」即可刪除該筆資料。
4. 若要清除全部購物車內容，可以按下「清空購物車」。
5. 系統會自動重新計算總金額。

---

### 3. 菜單管理流程

1. 進入「菜單管理」分頁。
2. 輸入飲料編號、飲料名稱、分類、基本價格與是否上架。
3. 按下「新增飲料」可新增資料。
4. 點選右側表格中的資料，可將資料帶入左側欄位。
5. 修改資料後按下「修改飲料」。
6. 選取資料後按下「刪除飲料」可刪除該筆資料。
7. 按下「儲存菜單」可將目前菜單資料寫入 `drinks.csv`。
8. 儲存後重新開啟程式，菜單資料仍會保留。

---

### 4. 歷史訂單查詢流程

1. 進入「歷史訂單」分頁。
2. 按下「顯示全部」可查看所有訂單。
3. 選擇日期後按下「依日期查詢」，可查詢指定日期的訂單。
4. 輸入關鍵字後按下「關鍵字查詢」，可搜尋訂單。
5. 可搜尋的關鍵字包含飲料名稱、甜度、冰塊、加料或訂單編號。
6. 系統會顯示今日訂單數與今日營業額。

- 使用 Windows Forms 製作圖形化介面。
- 使用 CSV 檔案完成資料讀寫。
- 具備點餐、購物車、菜單管理、歷史訂單查詢與營業額統計功能。
- 操作方式直覺，適合一般使用者操作。
- 菜單資料可以自行新增、修改、刪除。
- 歷史訂單可以依日期或關鍵字查詢。
- 完成訂單後可自動產生訂單編號並儲存紀錄。
